# Superstore_Data_Analysis-using-Python-Numpy-Pandas-Matplotlib-Seaborn-

This project focuses on analyzing Superstore sales data using Python to uncover important business insights such as total sales, profit trends, and category performance.
The analysis includes data cleaning, exploratory data analysis (EDA), and data visualization using popular Python libraries.
This project helps understand customer purchasing patterns, product category performance, and profitability, which are useful for business decision-making.

Technologies Used:
Python
Pandas – Data manipulation and analysis
NumPy – Numerical operations
Matplotlib – Data visualization
Seaborn – Statistical data visualization
Jupyter Notebook/ Google colab

The dataset used in this project is Superstore Sales Data, which contains information about:
Order Date
Ship Date
Product Category
Sales
Profit
Customer details
Order information

Data Cleaning
The following data preprocessing steps were performed:
